# 🤝 Contributing to **Sunlight Epstein Evidence**

This repository is for **factual, ethical, and legal** documentation.

---
## **📌 How to Contribute**
### **1. Submitting Evidence**
- **Only public records** (court filings, news articles, official reports).
- **Do NOT submit:** Illegal data, unredacted personal info, opinions, or conspiracy theories.
- Submit via **Pull Request (PR)**:
  1. Fork the repo.
  2. Create a new branch (`git checkout -b evidence/your-findings`).
  3. Add files to the correct folder (`/legal`, `/timelines`, `/networks`, `/media`).
  4. Include in PR: Source link, description, date.

### **2. Organizing Data**
- Filenames: Clear and descriptive (e.g., `2005-03-14_epstein_flight_logs.pdf`).
- Folders: Place files in the correct category.
- Formats: Prefer PDF, Markdown, or JSON.

### **3. Network Maps**
- Use **Mermaid.js** syntax for connection diagrams.

---
## **⚠️ Ethical & Legal Rules**
- **No doxxing.** Redact all personal info.
- **No illegal content.** Public records only.
- **No defamation.** Stick to verifiable facts.

---
## **✅ PR Checklist**
- [ ] Evidence is **public and legal**.
- [ ] Personal info is **redacted**.
- [ ] Sources are **cited**.
