# 🌍 **SUNLIGHT EPSTEIN EVIDENCE**
*Shining light on the shadows.*

---
### **🔦 What It Is**
A **centralized, searchable repository** of documents, timelines, and connections related to the Epstein case.
**No spin. Just facts.**

---
### **📂 What's Inside**
- **Legal Documents:** Court filings, depositions, and indictments.
- **Timelines:** Key events, dates, and individuals.
- **Network Maps:** Visualizations of connections (people, companies, locations).
- **Media Archives:** News clips, interviews, and primary sources.

---
### **🛠️ Tech Stack**
- **Format:** Markdown + JSON (for structured data)
- **Tools:** Python (for scraping/parsing), GitHub Pages (for hosting)
- **Visualization:** Mermaid.js (for network diagrams)

---
### **🚀 How to Use**
1. **Browse:** Navigate folders by category (e.g., `/legal`, `/timelines`).
2. **Search:** Use GitHub's search or `Ctrl+F` for keywords.
3. **Contribute:** Submit new evidence via PRs (see below).

---
### **📌 Key Folders**
```
📁 /legal/               # Court documents
📁 /timelines/           # Chronological events
📁 /networks/            # Connection maps (Mermaid diagrams)
📁 /media/               # News clips, images
📁 /raw/                 # Unprocessed data (for contributors)
```

---
### **🔍 Example Query**
**Q:** *"Who were Epstein's known associates in 2005?"*
**A:** Check `/networks/2005-associates.md` or `/timelines/2005.md`.

---
### **🤝 Contribute**
- **Found something?** Open a PR with:
  - Source link (if digital)
  - Context/description
  - Category (legal/timeline/network/media)
- **Standards:**
  - **No opinions.** Stick to verifiable facts.
  - **No doxxing.** Redact personal info (emails, addresses).
  - **Cite sources.** Always.

---
### **⚠️ Ethical Guidelines**
- **Do:** Share **publicly available** documents.
- **Don't:** Post illegal or unredacted private data.
- **Why:** Transparency **≠** vigilantism.
